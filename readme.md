# Contact Manager CLI (Python)

A simple command-line based Contact Manager application written in Python.

## Features
- Add new contacts
- View all contacts
- Search contacts by name or phone number
- Update existing contacts
- Delete contacts
- Input validation and error handling

## Tech Stack
- Python 3

## Future Improvements
- Persistent storage using SQLite
- Search optimization
- Flask-based web version

## How to Run
```bash
python main.py
